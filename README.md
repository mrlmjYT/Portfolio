# Personal Portfolio Website

A clean, responsive single-page portfolio designed to showcase my journey, skills, and future projects in web development and game design.

---

## Project Goal

The main objective of this portfolio is to create a central hub for my professional growth. It is built to:
- Showcase interactive games and websites I have developed.
- Serve as a digital resume for future career opportunities.
- Demonstrate clean and efficient frontend development practices.

## Tech Stack & Architecture

To keep the project lightweight and fast, it is built using core web technologies:

* **HTML5:** Handles the structural foundation and semantic text content of the entire website.
* **CSS3:** Powers the visual styling, custom layouts, background designs, and interactive button animations.
* **JavaScript (Vanilla):** Powers the **Single-Page Application (SPA)** logic. By dynamically managing content visibility, it eliminates the need for multiple HTML files, ensuring a smooth and seamless user experience.

## Key Features

- **Single-Page Layout:** Fast navigation without tedious page reloads.
- **Project Showcase:** A dedicated gallery section tailored specifically for web apps and game projects.
