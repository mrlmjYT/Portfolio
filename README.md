# Personal Portfolio Website

This is a clean interactive Portfolio where i display my Projects


## Project Goal

The main objective of this portfolio is to display my work
- Showcase interactive games and websites I have developed.

## Tech Stack & Architecture

To keep the project lightweight and fast, it is built using core web technologies:

* **HTML:** Is the entire Website for Text and co
* **CSS:** Is used for the entire design
* **JavaScript:** Used for the single Pager

## Key Features

- **Single-Page Layout:** Fast navigation and it loads everything at the start
- **Project Showcase:** Used for demonstrating my projects
